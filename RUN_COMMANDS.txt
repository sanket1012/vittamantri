BEST WAY: RUN CONTINUOUSLY WITH WATCHDOG
----------------------------------------

The watchdog keeps these processes running:
1. Flask backend
2. Telegram bot
3. React dashboard

If any process crashes, the watchdog starts it again.

Start watchdog:

cd C:\Users\Sanket Waghmare\Desktop\Vittamantri_Bot\vittamantri
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\scripts\start_vittamantri_watchdog.ps1

Stop everything:

cd C:\Users\Sanket Waghmare\Desktop\Vittamantri_Bot\vittamantri
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\scripts\stop_vittamantri_watchdog.ps1

Check logs:

cd C:\Users\Sanket Waghmare\Desktop\Vittamantri_Bot\vittamantri
Get-Content .\logs\watchdog.log -Tail 30
Get-Content .\logs\flask.err.log -Tail 30
Get-Content .\logs\bot.err.log -Tail 30
Get-Content .\logs\vite.err.log -Tail 30

IMPORTANT NOTES
---------------

For Telegram bot only, backend and bot must run.
For dashboard, backend and frontend must run.
The watchdog works only while your laptop is on and Windows is running.
If laptop is off or internet is disconnected, Telegram bot cannot work.
